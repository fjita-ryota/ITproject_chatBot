# ITproject ChatBot

ユーザーの入力に応じて会話を行うチャットボットアプリです。  
シンプルなUIでメッセージのやり取りができ、基本的な会話ロジックを実装しています。
## 補足
本アプリは外部API（ChatGPT API）を利用しており、
現在はAPIの利用期間終了により正常に動作しない状態となっています。

開発当初はユーザー入力に応じて動的な応答を返す機能を実装していました。
外部APIを利用したアプリ開発の経験として、API連携の仕組みやデータの扱い方を学びました。 

## 制作背景
ユーザーとの対話を通じて情報提供を行う仕組みに興味を持ち、
チャット形式でのUIとロジックの実装を学ぶために制作しました。

## 使用技術
- HTML
- CSS
- JavaScript

## 主な機能
- ユーザー入力に対する応答生成
- チャット形式のUI表示
- メッセージ履歴の表示

## 工夫した点
- ユーザーが違和感なく会話できるよう、自然な応答フローを意識しました
- 入力と出力の流れをシンプルに整理し、使いやすいUIにしました
- チャット形式の表示を整え、視認性を意識しました

## 今後の改善点
